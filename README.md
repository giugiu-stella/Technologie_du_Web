## Lancement du Projet

### Côté Frontend

1.  cd projet_main/client
2.  rm -r node_modules (on désinstalle pour mieux réinstaller avec tous les updates)
3.  npm install -g npm-check-updates
4.  ncu -u
5.  npm update
6.  npm start

### Côté Backend

1.  cd projet_main/server/src
2.  npm install
3.  node index.js
